### Week 4 Achievements:

**Theoretical Knowledge:**
* ✅ Understand Share Responsibility Model on AWS
* ✅ Master IAM concepts: Users, Groups, Policies, Roles
* ✅ Learn about AWS Cognito for user authentication
* ✅ Understand AWS Organization and Identity Center
* ✅ Learn KMS (Key Management Service) and encryption
* ✅ Study AWS Security Hub and security best practices

**Practical Skills:**
* ✅ **Security Hub:** Enable and check security scores
* ✅ **IAM Management:** 
  - Create IAM users with appropriate permissions
  - Create IAM groups and assign users
  - Write IAM policies (JSON format)
  - Create IAM roles with trust relationships
  - Practice switch roles between accounts
* ✅ **IAM Best Practices:**
  - Compare IAM Role vs Access Key
  - Understand when to use role vs access key
  - EC2 instance profile with IAM role
  - Principle of least privilege

**Tools & Technologies:**
