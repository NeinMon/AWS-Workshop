### Kết quả đạt được tuần 4:

**Kiến thức lý thuyết:**
* ✅ Hiểu về Share Responsibility Model trên AWS
* ✅ Nắm vững khái niệm IAM: Users, Groups, Policies, Roles
* ✅ Biết về AWS Cognito cho user authentication
* ✅ Hiểu AWS Organization và Identity Center
* ✅ Nắm về KMS (Key Management Service) và encryption
* ✅ Học về AWS Security Hub và security best practices

**Kỹ năng thực hành:**
* ✅ **Security Hub:** Enable và check security scores
* ✅ **IAM Management:** 
  - Tạo IAM users với permissions phù hợp
  - Tạo IAM groups và assign users
  - Viết IAM policies (JSON format)
  - Tạo IAM roles với trust relationships
  - Practice switch roles giữa các accounts
* ✅ **IAM Best Practices:**
  - So sánh IAM Role vs Access Key
  - Hiểu khi nào dùng role, khi nào dùng access key
  - EC2 instance profile với IAM role
  - Principle of least privilege

**Tools & Technologies:**
* AWS IAM Console
* AWS Security Hub
* IAM Policy Simulator
* AWS CLI với IAM commands
* AWS Resource Groups & Tag Editor
* JSON Policy Editor
* Multi-factor Authentication (MFA) devices
