### Internship Program Overview

This internship program was completed over **12 weeks** (3 months) at First Cloud Journey, focusing on learning and applying AWS services to build a complete serverless Backend system.

#### **Phase 1: AWS Foundation (Week 1-3)**
- Getting familiar with AWS Cloud and basic services
- Learning about VPC, Networking and Security
- Studying Storage services (S3, FSx, Storage Gateway)

#### **Phase 2: AWS Fundamentals (Week 4-7)**
- Deep dive into IAM and Security best practices
- Learning about RDS and Database services
- Hands-on with DynamoDB and Serverless architecture

#### **Phase 3: Backend Development (Week 8-12)**
- Developing REST APIs with Lambda, API Gateway and DynamoDB
- Implementing Chat System with polling mechanism
- Production deployment and documentation completion
