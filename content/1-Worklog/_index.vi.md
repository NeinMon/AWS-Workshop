### Tổng quan chương trình thực tập

Chương trình thực tập được thực hiện trong vòng **12 tuần** (3 tháng) tại First Cloud Journey, tập trung vào việc học tập và áp dụng các dịch vụ AWS để xây dựng hệ thống Backend serverless hoàn chỉnh.

#### **Giai đoạn 1: Nền tảng AWS (Tuần 1-3)**
- Làm quen với AWS Cloud và các dịch vụ cơ bản
- Tìm hiểu về VPC, Networking và bảo mật
- Học về Storage services (S3, FSx, Storage Gateway)

#### **Giai đoạn 2: AWS Fundamentals (Tuần 4-7)**
- Tìm hiểu sâu về IAM và Security best practices
- Học về RDS và Database services
- Thực hành với DynamoDB và Serverless architecture

#### **Giai đoạn 3: Backend Development (Tuần 8-12)**
- Phát triển REST APIs với Lambda, API Gateway và DynamoDB
- Triển khai Chat System với polling mechanism
- Deploy production và hoàn thiện documentation
